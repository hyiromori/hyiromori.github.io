最近 PHP で開発しているのですが、Docker 上で動く PHPUnit を実行するために必要なプラグインを探すのに手間取ったので、備忘録。

たぶん、PHPStorm なら最初から揃ってるんだろう、と思う。

## 必要なプラグイン

以下の４つのプラグインがあれば設定できます。

* PHP
* PHP Remote Interpreter
* SSH Remote Run
* PHP Docker
